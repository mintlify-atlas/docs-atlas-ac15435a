      Get up and running with ClubConnect UK in minutes
    </p>
  </div>

  <Steps>
    <Step title="Set up your environment">
      Clone the repository and install dependencies:
      
      ```bash
      git clone https://github.com/yet-joshua/ClubConnectUK.git
      cd ClubConnectUK
      pnpm install
      ```
    </Step>
    
    <Step title="Configure Supabase">
      Create a `.env` file with your Supabase credentials:
      
      ```bash
      VITE_SUPABASE_URL=your_supabase_url
      VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
      ```
      
      Run the database migrations to set up your schema.
    </Step>
    
    <Step title="Start the development server">
      Launch the application locally:
      
      ```bash
      pnpm dev
      ```
      
      Navigate to `http://localhost:3000` to see the login page.
    </Step>
    
    <Step title="Explore the features">
      After logging in, you can:
      - View the dashboard with KPIs and upcoming events
      - Manage clubs and schools
      - Schedule teacher covers
      - Send broadcast messages
      - Use the interactive calendar
    </Step>
  </Steps>
</div>
